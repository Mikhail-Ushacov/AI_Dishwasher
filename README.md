AI_Dishwasher

AI_Dishwasher is a kitchen chaos simulator starring an AI agent.
The agent completes kitchen orders and gradually learns how to do them faster, smarter, and with fewer unnecessary trips.

The kitchen is modeled as a graph:

nodes represent workstations (stove, sink, table, etc.),
edges represent the time required to move between them.

At first, the AI behaves like a clumsy intern, wandering around and wasting time on simple tasks.
Over time, it learns to optimize routes, make better decisions, and understand that every extra step has a cost.

AI_Dishwasher is about:

learning by doing,
path optimization in a kitchen environment,
a pinch of algorithms,
and a lot of spilled virtual soup.
